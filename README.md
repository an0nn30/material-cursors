# Material Cursors

### Manual Install

1. Install dependencies:

    - git
    - make
    - inkscape
    - xcursorgen

2. Run the following commands as normal user:

    ```
    git clone https://github.com/varlesh/material-cursors.git
    cd material-cursors
    make build
    sudo make install
    ```

3. Choose a theme in the Settings or in the Tweaks tool.
